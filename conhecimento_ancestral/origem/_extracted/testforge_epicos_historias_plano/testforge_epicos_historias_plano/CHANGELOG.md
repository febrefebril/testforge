# Changelog

## [0.2.0] - 2026-06-12

### Added

- Plano inicial de épicos e histórias.
- Template de critérios de aceite.
- Guia de versionamento do plano macro.
- Backlog em YAML.

- Política do EvidenceCollector definida como `alert_only` para dados sensíveis no MVP.

### Removed

- Mascaramento automático de CPF/dados sensíveis no EvidenceCollector para o MVP.

## [0.1.0] - 2026-06-12

### Added

- Plano macro inicial do TestForge.
- Conceitos de MIS, shadow mode, Promotion Gate, EvidenceCollector e Synthetic Lab.
